# 簡易財報解讀指南 (Simplified Financial Reading Manual)

本系統致力於將專業的財務語言轉化為大眾易懂的商業直覺。以下是系統解讀指標的邏輯標準：

## 1. 賺錢的品質與效率

*   **產品好不好賺 (毛利率)**:
    *   `> 40%`: **高**。具備獨特競爭力或品牌溢價。
    *   `20% - 40%`: **中**。處於競爭市場，但有穩定獲利。
    *   `< 20%`: **低**。代工、零售或競爭極其激烈的紅海產業。

*   **賺錢效率 (利益率)**:
    *   **增長**: 公司營運效率提升，或開支控管良好。
    *   **下降**: 警示訊號，代表營運成本增加或管銷費用失控。

*   **投資回報率 (ROIC)**:
    *   `> 15%`: **優異**。公司像一台強大的賺錢機器。
    *   `< 借錢成本 (WACC)`: **警示**。投資越多虧越多，正在毀滅股東價值。

## 2. 現金流與體質

*   **口袋裡的現金 (自由現金流)**:
    *   **正值**: 公司有實力發股息或再投資，不需看銀行臉色。
    *   **負值**: 警示訊號。公司入不敷出，可能面臨融資壓力。

*   **欠別人的錢 (負債)**:
    *   **高槓桿**: 若負債比 > 70% 且利息保障倍數低，風險極大。

## 3. 智能體溝通規範

1.  **數字與文字併行**: 「毛利率為 45% (高)」優於單純寫 45%。
2.  **避免艱澀術語**: 用「口袋裡的錢」代替「FCF」，用「賺錢效率」代替「EBIT Margin」。
3.  **情境類比**: 「目前公司營運就像是在打八折出清」比「估值低於 1 個標準差」更容易理解。
