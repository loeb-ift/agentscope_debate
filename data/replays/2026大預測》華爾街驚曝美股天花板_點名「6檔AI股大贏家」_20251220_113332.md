### 3.2 敏感度分析

| 變數 | 變動幅度 | 預期影響 | 公式示例 |
|------|----------|----------|----------|
| **AI 需求增長** | -20% | NVIDIA EPS 8.3 → 6.7 元（-19%） | EPS_new = EPS_base × (1 – 0.20) |
| **新增股發行** | +30% 於市值 | EPS 下修 25%（假設每股分母增加 30%） | EPS_new = EPS_base × (1 – 0.25) |
| **監管成本上升** | +15% | Alphabet 股價目標 4000 → 3400 美元（-15%） | PV_new = PV_base × (1 – 0.15) |
| **VIX 上升** | +5% | 市場波動性升高，AI 股回報波動加大 10% | σ_new = σ_base × 1.10 |

> **結論**：正方的「天花板」預測在最佳情境下可達 12.3% CAGR，但若上述敏感變數發生負面變動，整體回報將顯著下滑。反方提出的稀釋與監管成本正是上述敏感度模型中的關鍵負面因子。

### 3.3 邏輯攻防

| **正方** | **反方** | **互動關係** |

## 2.  Pull the holdings programmatically

Below is a **stand‑alone Python 3 script** that:

1. Pulls ARKK’s holdings from the ARK website (HTML scraping).  
2. Returns a `pandas.DataFrame` with ticker, company name, and weight.  
3. Compares the extracted tickers to your “AI‑core” list (NVDA, MSFT, GOOGL, AMZN, TSLA, AAPL).  

```python
#!/usr/bin/env python3
# ----------------------------------------------------------
#  1. Install deps first (one‑time):
#     pip install requests beautifulsoup4 pandas
# ----------------------------------------------------------
import requests
from bs4 import BeautifulSoup
import pandas as pd

# 1️⃣ URL of ARKK holdings page
URL = "https://www.ark-invest.com/ark-funds/ark-innovation-fund-holdings/"

# 2️⃣ Pull HTML
r = requests.get(URL)
r.raise_for_status()            # fail fast if 404/500

# 3️⃣ BeautifulSoup parse
soup = BeautifulSoup(r.text, "html.parser")

# 4️⃣ Find the holdings table – ARK uses a <table id="holdings">
table = soup.find("table", {"id": "holdings"})

# 5️⃣ Read into pandas
df = pd.read_html(str(table))[0]

# 6️⃣ Clean column names (they may contain whitespace/newlines)
df.columns = [c.strip() for c in df.columns]

# 7️⃣ Show the top‑10 (for sanity)
print(df.head(10))

# 8️⃣ Build a quick comparison
ai_tickers = {"NVDA", "MSFT", "GOOGL", "AMZN", "TSLA", "AAPL"}
df['Ticker'] = df['Ticker'].str.upper()

# 9️⃣ Which AI tickers are present?
present = df[df['Ticker'].isin(ai_tickers)]
print("\n=== AI tickers found in ARKK holdings ===")
print(present[['Ticker', 'Company', 'Weight (%)']])

# 10️⃣ Which AI tickers are missing (unlikely but handy)?
missing = ai_tickers - set(present['Ticker'])
print("\n=== AI tickers NOT present ===")
print(missing if missing else "All 6 AI tickers are in ARKK")

```
